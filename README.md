About the BlogApp
- Home page contains counter for demonstartion of contextProvider
- Header contains simple nav bar for home, create employee and view list of employees
- createEmp allows us to add details of employee and add them to the list
- listOfEmp shows all the employee details and allows to view edit or delete a particular employee details
